EchoShade: Escape the Shadow

EchoShade: Escape the Shadow is a simple survival game developed in C. The objective of the game is to survive for as long as possible by avoiding moving enemy blocks inside a grid-based arena while maintaining a high score.

About the Project

I built this project to improve my understanding of game development concepts such as player movement, collision detection, score management, timers, and real-time rendering.

The game includes a start screen, gameplay screen, score and timer system, and a game over screen with restart functionality.

Features

• Player movement using keyboard controls
• Moving enemy blocks
• Collision detection
• Score tracking system
• Countdown timer
• Start screen and Game Over screen
• Restart option without restarting the application

Technologies Used

• C Programming
• Raylib Graphics Library
• Code::Blocks IDE

Controls

Key| Action
Enter| Start Game
Arrow Keys| Move Player
R| Restart Game

How to Run

1. Open the project in Code::Blocks.
2. Build and run the project.
3. Press Enter to start the game.
4. Use the arrow keys to move and avoid obstacles.
5. Try to achieve the highest score before the game ends.
